# AI Multilingual Platform

## Run the app

Use the generated launcher:
```bat
run_app.bat
```

Or manually from PowerShell:
```powershell
cd "c:\Users\KIIT\OneDrive\Desktop\AI Multilingual Platform"
& ".\venv\Scripts\python.exe" -m streamlit run app.py
```

## Project layout

- `app.py` - root Streamlit launcher
- `frontend/app.py` - actual Streamlit application
- `backend/` - translation, TTS, language detection, and speech services
- `scripts/` - helper/demo scripts
- `tests/` - test files
- `outputs/` - generated audio/video files
- `venv/` - Python virtual environment

## Notes

- Ensure the virtual environment is activated or use the full `python.exe` path above.
- Open `http://localhost:8501` after startup.
